
# StarCoin

区块链demo

## 启动节点

```sh
cd c:\myfile\program\btc\starcoin ; $env:PORT="3000" ; node server.js
cd c:\myfile\program\btc\starcoin ; $env:PORT="3001" ; node server.js
```

## 已完成

- 可视化。用一个html网页实现可视化。在一个html网页上，可以看到所有节点的信息。

- 实现多节点功能，完成两个节点的版本。连接节点后，让所有节点的情况相同。允许主动断开和某个其他节点的连接。

- 支持数据保存到本地。
